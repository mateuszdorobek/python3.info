Scatter Plot
============


Important
---------
* Used to show correlation

Syntax
------
.. code-block:: python

    import matplotlib.pyplot as plt


    x = [1, 2, 3, 4]
    y = [1, 2, 3, 4]

    plt.scatter(x, y)
    plt.show()


Simple Points
-------------
.. code-block:: python

    import matplotlib.pyplot as plt
    import numpy as np


    x = np.linspace(0, 10, 50)
    y = np.sin(x)

    plt.scatter(x, y)
    plt.show()


Markers
-------
* ``marker``
* ``s`` - size

.. figure:: img/matplotlib-markers-filled.png

.. figure:: img/matplotlib-markers-unfilled.png

.. code-block:: python

    import matplotlib.pyplot as plt


    x = [1, 2, 3, 4, 5, 6, 7, 8]
    y = [5, 2, 4, 2, 1, 4, 5, 2]

    plt.scatter(x, y, marker='*', s=100)
    plt.show()

.. code-block:: python

.. code-block:: python

    import matplotlib.pyplot as plt
    import numpy as np


    x = [1, 2, 3, 4, 5, 6, 7, 8]
    y = [4, 7, 6, 2, 7, 4, 5, 2]

    plt.scatter(x, y, marker='*', s=500)
    plt.scatter(y, x, marker='o', s=500)

    plt.show()
