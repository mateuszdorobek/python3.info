Create About
============
* Scalar - 0 dimensional array
* Vector - 1 dimensional array
* Matrix - 2 dimensional array
* Tensor - 3 dimensional array


Scalar
------
* Scalar - 0 dimensional array

.. code-block:: python
    :caption: Scalar

    1


Vector
------
* Vector - 1 dimensional array

.. code-block:: python
    :caption: Vector

    [1, 2, 3]


Matrix
------
* Matrix - 2 dimensional array

.. code-block:: python
    :caption: Matrix

    [[1, 2, 3],
     [4, 5, 6],
     [7, 8, 9]]


Tensor
------
* Tensor - 3 dimensional array

.. code-block:: python
    :caption: Tensor

    [[[1, 2, 3],
      [4, 5, 6],
      [7, 8, 9]],

     [[1, 2, 3],
      [4, 5, 6],
      [7, 8, 9]]]
